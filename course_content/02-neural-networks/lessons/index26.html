Your Neural Network Construction Kit

Imagine you have a toolkit with the basic building blocks of a neural network. For this experiment, your kit contains:

  • Input Nodes: These represent where your data will enter.
  • Hidden Neurons: These are the processing units in the middle layers.
  • Output Neurons: These will produce the network's final prediction.
  • Connection Tool: To draw the pathways (weights) between them.

Remember, in a feedforward network, information flows from input to output, without looping back. In a fully connected layer, every unit in one layer connects to every unit in the next layer.

Stop and Think

In the 2-input, 3-hidden-neuron, 1-output-neuron network you just built, how many weight parameters would there be in total (not counting biases)?

Let's count:
- Connections from 2 inputs to 3 hidden neurons: Each of the 3 hidden neurons gets 2 inputs, so $$2 \times 3 = 6$$ weights.
- Connections from 3 hidden neurons to 1 output neuron: The 1 output neuron gets 3 inputs, so $$3 \times 1 = 3$$ weights.
Total weights = $$6 + 3 = 9$$ weights.

Task 2: Counting Connections (Parameter Awareness)

Now that you're getting the hang of connecting things, let's focus on the number of connections (which correspond to weight parameters).

This exercise helps you start thinking about the number of parameters (weights) in a network. Even for small networks, this number can grow quickly! Imagine a network with hundreds of neurons per layer – that's a lot of weights for the network to learn!

Why It Matters

The number of learnable parameters (weights and biases) in a network is directly related to its 'capacity' – its ability to learn complex functions. More parameters generally mean more capacity, but also a higher risk of overfitting if you don't have enough data, and longer training times.

Frequently Asked Questions

In these simple networks, how many bias parameters would there be?

Good question! Generally, each processing neuron (i.e., neurons in the hidden layers and the output layer, but not the input nodes) has its own individual bias term. So, in the network from Task 1 (2 inputs, 3 hidden, 1 output), there would be 3 bias parameters for the hidden layer and 1 bias parameter for the output layer, totaling 4 bias parameters.
